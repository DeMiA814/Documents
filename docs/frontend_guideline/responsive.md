# レスポンシブ対応について

## デザイン領域において
XDなどでのアートボードの選択に活用すること
- スマホは、360x640 px
- タブレットは、834 x 1194px
- デスクトップは、1280 x 720px
## ブレイクポイント（BP）について
- スマホとデスクトップの2つに分ける場合、BPはwidth:1024px
- スマホとタブレットとデスクトップの3つに分ける場合、BPはwidth:600と1024
- さらに刻む場合は、840も導入する

## ポイント
小さい幅に合わせたCSSを大きい幅に引き伸ばすとき、崩れは少ないが、その逆は、崩れやすい。よって、小さい幅に合わせたCSSをつける。
